# Ghost Window - Product Overview

## Purpose
Ghost Window — это небольшая системная утилита для Windows, которая позволяет пользователям временно делать любое окно под курсором мыши прозрачным и "прозрачным" для кликов (click-through). Это позволяет взаимодействовать с окнами, находящимися позади него, без необходимости сворачивать или перемещать верхнее окно.

## Problem Solved
- **Многооконная работа**: При работе с несколькими окнами часто нужно временно "заглянуть" или взаимодействовать с окном позади текущего
- **Сравнение данных**: Необходимость видеть содержимое одного окна при работе в другом
- **Минимизация переключений**: Избежание постоянного Alt+Tab для переключения между окнами

## User Experience Goals

### Primary Interaction
- **Удержание Win (LWin) > 0.5 сек**: Окно под курсором становится прозрачным и click-through
- **Отпускание Win**: Окно мгновенно возвращается в нормальное состояние
- **Короткое нажатие Win (< 0.5 сек)**: Стандартное поведение — открытие меню Пуск

### Feedback Mechanisms
- **Визуальная**: Полупрозрачное окно (80/255 = ~30% непрозрачности)
- **Текстовая**: Всплывающая подсказка "👻 Ghost Mode" рядом с курсором
- **Звуковая**: Короткий звуковой сигнал (Beep 1000Hz при активации, 500Hz при деактивации)

### Safety Features
- Окна "Progman", "WorkerW", "Shell_TrayWnd" (рабочий стол и панель задач) игнорируются
- Single instance — приложение запускается только в одном экземпляре
- Автоматическое восстановление окна при отпускании клавиши или ошибке

## Target Users
- Продвинутые пользователи Windows
- Разработчики и IT-специалисты
- Пользователи, работающие с множеством окон одновременно

## Platform Requirements
- **OS**: Windows 10 / 11
- **Architecture**: x64 (использует .NET Framework 4.0)
- **Permissions**: Требуются права для установки глобального хука клавиатуры
