# Проект: Ghost Window (Сквозной просмотр окон)

## Описание
Небольшая утилита для Windows Она позволяет временно сделать активное окно под курсором прозрачным и "проницаемым" для кликов мыши (Click-through), чтобы взаимодействовать с окнами, находящимися позади него.

## Функционал
*   **Удержание клавиши `Win` (LWin) более 0.5 секунд:** 
    *   Окно под курсором становится прозрачным (уровень 80/255).
    *   Окно становится "призрачным" (клики проходят сквозь него).
    *   Появляется всплывающая подсказка "👻 Ghost Mode".
    *   Раздается звуковой сигнал.
*   **Отпускание клавиши `Win`:**
    *   Окно мгновенно возвращается в исходное состояние (непрозрачное, кликабельное).
*   **Короткое нажатие `Win` (менее 0.5 секунд):**
    *   Работает стандартно (открывается меню «Пуск»).

## Требования
*   ОС: Windows 10 / 11.
