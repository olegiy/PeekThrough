# Ghost Window - Текущий контекст

## Recent Changes

### Последние изменения
- Добавлена поддержка множественных окон в Ghost Mode
- Добавлен метод `DeactivateGhostMode()` для внешней деактивации
- Добавлено свойство `IsGhostModeActive` для проверки состояния
- Рефакторинг `KeyboardHook.cs` для отслеживания других клавиш
- Добавлен `DebugLogger.cs` для логирования отладочной информации
- Теперь Ghost Mode автоматически деактивируется при нажатии любой другой клавиши
- Блокировка активации если другие клавиши нажаты до Win

## Известные проблемы
- Некоторые UWP/Modern приложения могут некорректно обрабатывать WS_EX_TRANSPARENT
- Для некоторых окон может потребоваться запуск от администратора
- Окна, изначально имеющие WS_EX_LAYERED, могут сохранить прозрачность после восстановления
